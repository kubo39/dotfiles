# dotfiles

my set of configuration files.

## Use

```
curl -SsL https://raw.githubusercontent.com/kubo39/dotfiles/master/setup.sh | sh
```

or

```
git clone git@github.com:kubo39/dotfiles ~/dotfiles
sh ~/dotfiles/setup.sh
```
